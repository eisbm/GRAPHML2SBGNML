## 1. PROJECT

Allow convertion of yEd GraphML files from and to SBGN-ML format.

## Requirements

Java 8 SDK
Maven

## 2. INSTALL

Clone the repository.

`mvn clean install`

JAR files will be put into a `dist` folder at the root.

To generate the javadoc:

`mvn site`

Javadoc is placed into the `docs` folder.
